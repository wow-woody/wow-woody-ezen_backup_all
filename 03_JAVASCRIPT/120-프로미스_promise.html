프로미스 promise

자바스크립트의 비동기처리 방식 중 하나로,

작업한 결과물이 성공할 수 있고, 실패할 수도 있는 상황을 다룰 때

        new promise((resolve,reject)=> {
            if(성공!){
                resolve('작업 성공');
            }; 
            else {
                reject('작업 실패');            
            };
        });
        변수.then (result => {
            //작업 성공
        }).catch(error => {
            //작업 실패
        })
