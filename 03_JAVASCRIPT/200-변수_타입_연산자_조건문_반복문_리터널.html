변수

변할 수 있는 데이터를 저장하는 공간

        var(variable) - 이전에 사용 : 재선언 및 재할당 가능
        let : 다시 사용할 수 없음. 재선언 불가능 재할당 가능
        const :  다른 프로그램에서 변할 수 없는 데이터를 저장하는 공간

var 변수명1 = 데이터; let 변수명2; 변수명1 = null

데이터 타입

        숫자(Number) : 1, 2.5
        문자(String) :"", ''
        블리언(boolean) : 참 또는 거짓 - true or false   1-참, 0-거짓, 조건문에 주로 사용

        null 의도적으로 공간을 비운 경우 또는 값이 없음, 존재하지 않음
        undefined  : 변수 선언을 할 때 데이터를 할당하지 않음 즉 값이 미정 또는 값이 할당되지 않은 상태
        BigInt : 64bit

연산자

데이터를 조작하여 처리하는 방식을 뜻함

        산술 연산자 : +, -, *, /, %
        연결 연산자 : + (잘 안씀) ->   `백틱` 템플릿 리터널 방식
        증감 연산자 : ++, --
        대입 연산자 : =, +=, -=, *=, /=. %=
        비교 연산자 : >, >=, <, <=. ===. !=   -> true, false
        논리 연산자 : &&(and), ||(or), !(not)
        삼항 연산자 : 조건문? 조건이 참일 때 실행 : 조건이 거짓일 때 실행    -> A?A:B

조건문

조건문의 참, 거짓을 확인해서 처리하는 명령어

if문 (if, if~else, if ~ else if ~ else)

        if(조건문) {
            조건의 결과가 참일 때 실행항 문장;
        }


        if(조건문) {
            조건의 결과가 참일 때 실행항 문장;
        }
        else {
            조건의 결과가 거짓일 때 실행항 문장;
        }


        if(조건문 1) {
            조건의 결과가 참일 때 실행항 문장;
        }
        else if(조건문 2) {
            조건문1의 결과가 거짓일 때 실행항 문장;
        }
        else if(조건문 3) {
            조건문2의 결과가 거짓일 때 실행항 문장;
        }
        ....
        else {
            모든 조건문이 거짓일 때 처리할 문장;
        }

swich 문 (swich~case) : 조건이 많은 경우

        swich (조건에 대한 값을 저장한 변수) {
            case "값1":
                값1을 처리할 문장; 
            breakk;   
            case "값2":
                값2을 처리할 문장;   
            breakk;    
            default: 위에 닶이 없는 경우 마지막에 실행해서 처리;
        }

반복문

같은 명령어를 여려번 실행해야될 경우

for : 정해진 횟수 만큼 반복 while : 조건이 맞을 때까지 반복 do~while : 조건과 상관없이 무조건 한번 싱행 

        for(초기값; 조건문; 증감식) {
            반복해서 실행할 문징;
        }
