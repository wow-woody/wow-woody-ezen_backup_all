함수

자주 사용되는 명령들을 한곳에 묶어두는 명령어

1. 선언적 함수(명명함수) : 이전 ***호출하지 않으면 표현, 구현, 구동되지 않음

2. 함수 표현식(익명함수)

3. 화살표 함수 es6부터 도입, 합수를 간결하게 사용하기 위해 도입 this는 사용불가

호이스팅(Hoisting) : 선언 전에 호출 가능, 변수나 함수 선인이 마치 코드의 최상단으로 끌어올려진 것 처럼 동작하는 현상_오류뜸?


        ✅
        1. 선언적 함수 표현식
        function 함수이름(매개변수) {
            실행할 명령어 및 값;
        }

        ✅✅
        2. 함수 표현식
        let 함수이름 = function (매개변수) {
            실행할 명령어 및 값  ;  
        }

        ✅✅✅
        3. 화살표 함수 표현식
        let 함수이름 = (매개변수) => {
            실행할 명령어 및 값;
        }
        let 함수이름 = (매개변수, 매개변수2) => {
            실행할 명령어 및 값;
        }

        **매개변수: 함수를 정의할 때 함수 내부에서 사용될 값의 이름을 붙이는 변수
                    - 함수 안 또는 함수 매서드에 넣은 변수
