제어문

프로그램의 흐름을 제어할때 사용하는 명령어를 뜻한다

조건문, 반복문, 보조제어문이 있다

조건문

조건에 따라서 프로그램의 흐름을 제어할때 사용한다

if, switch

if

if 비교연산,조건이 적은 경우

단일 if

if~else

다중 if

switch

값을 비교,조건이 많은 경우

        switch(변수명){
            case 값1:
            값1이 맞으면 실행할 문장;
            break;
            case "값2":
            값2 가 맞으면 실행할 문장;
            break;
            default:
            위의 모든 값이 아닌경우 처리할 문장;
        }
